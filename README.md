# Конвентор цветов

## Описание
Цветовой конвентер — это веб-приложения для определения значений разлчиных цветовых моделей для одного и того же цвета. Веб-приложение предоставляет возможность работать с такими моделями, как RGB, CMYK и XYZ.
## Файлы проекта
### `main.html`
Этот файл отвечает за структуру и объекты веб-приложения. Здесь описывается ввод через ползунки и поля ввода, выбор цвета при помощи кнопок и цветовой палитры, а также выскакивающее окно. Одним слово, фундамент всего проекта.
### `style.css`
Данный файл отвечает за дизайнерскую составляющую веб-приложения. Здесь описываются:
- панель функциональна и её форма;
- размер кнопок и их цвета;
- положение всех объектов;
- вид ползунков;
- стиль выскакивающего окна.
### `code.js`
Последний же файл содержит весь фукционал страницы. Тут описываются синхранизация различных цветовых моделей (подробнее о них описано ниже), синхранизация поле ввода с ползунками, различные ограничения, а также многое другое.
## Цветовые модели
- **RGB**:
    - Состоит из трёх составляющих: красный цвет (0-255), зелёный цвет (0-255), синий цвет (0-255). Это аддитивная цветовая модель, описывающая способ кодирования цвета для цветовоспроизведения с помощью трёх цветов, которые принято называть основными. Выбор основных цветов обусловлен особенностями физиологии восприятия цвета сетчаткой человеческого глаза.
    - В моём проекте именно она является основной. Встречается в функциях таких, как `hexToRgb`, `cmykToRgb`, `xyzToRgb` и им обратные.
- **CMYK**:
    - Это субтрактивная схема формирования цвета, а также четырёхцветная автотипия, используемая прежде всего в полиграфии для стандартной триадной печати или печати триадными красками. Она использует голубой (0-100), пурпурный (0-100) и жёлтый (0-100) цвета в роли основных, а также чёрный цвет (0-100).
- **XYZ**:
    - Концепция трехцветных значений XYZ основана на идее видеть цвета, и гласит, что глаз состоит из трех типов фоторецепторов, которые позволяют видеть три основных цвета (красный, зеленый, синий), в то время как все цвета являются смесями трех основных цветов. Поэтому цвет описан в трехцветных координатах X,Y,Z.
- **HEX**:
    - Является вспомогательной моделью. Для глаза пользователя она не видна, но используется для использования в `style.css` при переводе с других цветовых моделей.
## Работа проекта
Для ввода данных о цвете есть возможность:
- Выбор цвета при помощи **ползунков**.
- Ввод при помощи текстовых **полей**.
- При помощи **"цветовой кнопки"**, расположенной в нижней части панели.
## Как запустить проект
1. Скачайте репозиторий и запустите файл `main.html` или перейдите по ссылке https://ddjoul.github.io/pkgFirstLab/main.html
2. Используйте текстовые поля, ползунки или цветовой выбор для управления цветами.
3. Цвет автоматически поменяется в каждой цветой модели (при невозможности появится ошибка). Сам цвет можно увидеть как на "цветовой кнопке", так и на заднем фоне.
## Используемые технологии
- **HTML5**. Для создания структуры веб-приложения.
- **CSS3**. Для описания лизайна, а также для созлания пользовательского интерфейса.
- **JavaScript**. Для описания функций перевода цветов и их использования в проекте.
